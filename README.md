# Real Estate App

Live web app for finding properties near you with interactive map search.

## Features
- 🗺️ Map-based property discovery
- 📍 Real GPS location-based search
- 🏠 Property listings with details
- 💬 Direct contact with owners
- ✅ Verified properties

## Tech Stack
- HTML, CSS, JavaScript
- Leaflet.js for maps
- Responsive design (mobile-first)
