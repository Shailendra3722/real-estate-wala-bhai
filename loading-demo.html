Premium Loading Animation

1. Manual Show/Hide

Display the loading animation manually and hide it after a delay.

PremiumLoader.show();
// ... do something ...
PremiumLoader.hide();

2. Page Transition

Show loading animation while navigating to another page.

PremiumLoader.showForTransition('home.html');

3. Wrap Async Operation

Show loading while performing an async operation (e.g., API call).

await PremiumLoader.wrapWithLoader(async () => {
  // Your async code here
  const data = await fetch('/api/properties');
  return data;
});

4. Transition Mode (Faster)

Use faster animation for quick page transitions.

PremiumLoader.show(true); // true = transition mode

📋 Integration Guide

Add these lines to the <head> of your HTML pages:

<link rel="stylesheet" href="loading-animation.css">
<script src="loading-animation.js"></script>

The loader will automatically show on page load and hide when ready!
